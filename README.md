# PROJECT-HR-Attrition-Dashboard
I Built an interactive HR Attrition Dashboard in Microsoft Excel to analyze employee turnover patterns across an organization.

What I did:
Analyzed attrition trends by department, age group, job role, and salary band
Used PivotTables, Pivot Charts, and advanced Excel formulas for data processing
Designed a clean, visual dashboard layout for easy reporting and insight delivery.

Key Insight: Identified high-attrition segments to help HR teams make data-backed retention decisions.

Tools Used: Microsoft Excel | Power Query | PivotTables | Data Visualization | Dashboard Design
